## Findings considered and rejected (do not re-audit)

- **Magic-link verify TOCTOU race**: by design — the single-use burn
  (`UPDATE ... WHERE used_at IS NULL`, backend/src/routes/auth.ts:101-108)
  gates session/user creation; concurrent verifies lose cleanly.
- **buyer-fit "dead wire"**: false — `evaluateFit` is wired
  (lib/buyer-report/build.ts:185) and rendered
  (components/buyer/BuyerReportPanel.tsx:282-307).
- **generatedAt re-stamp churn**: false — apply scripts preserve the incoming
  `generatedAt` (e.g. scripts/apply-civic.ts:73,102). Single-line JSON is a
  deliberate size choice; data review happens via the coverage gate, not git
  diff.
- **Rate-limiter eventual-consistency window**: by design and documented in
  backend/src/lib/rate-limit.ts (throttle, not a hard boundary).
- **Rename MelbourneMap → CityMap**: cosmetic; region prop is visible;
  not worth the churn now.
- **@types/unzipper removal**: trivial, unverified benefit; fold into a
  future hygiene pass if typecheck proves it unused.
- **O(n²) overlay joins (normalize.ts)**: real but bake fits the 180-min CI
  window today; revisit when bake time actually hurts (record timing first).
- **POI 7.5MB split/zoom-gating**: real UX cost on first toggle, but the
  essential/extended split is a product decision — parked for the founder.
- **State-management consolidation**: do it with the accounts phase, not
  before (prop-drilling is annoying but tested and stable).
- **GitHub Actions SHA-pinning**: standing maintenance cost > current risk
  for a public repo; revisit at commercial launch.
- **Tailwind 4 / vitest 4 / lucide 1.x / csv-parse 6 majors**: no forcing
  function; each needs its own migration window.
- **ASGS Edition 4 (July 2026)**: not a plan — a WATCH item. Pipeline pins
  ASGS2021 endpoints deliberately (NATIONAL-ROLLOUT.md); migration is its own
  future project after ABS publishes Ed.4 services.
